Html

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).

Css

"Cascading Style Sheets" o "Hojas de Estilo en Cascada",Usado para definir la presentación visual de una página web, separando el contenido del diseño. Permite controlar elementos como colores, fuentes, tamaño, espaciado, entre otros, para crear una apariencia visual consistente y atractiva.

JavaScript

es la extensión de archivo más común para los archivos de código JavaScript. Estos archivos contienen instrucciones que los navegadores web interpretan para crear páginas web interactivas y dinámicas. Los archivos .js son esencialmente archivos de texto plano que pueden ser editados con cualquier editor de texto y se ejecutan en el navegador del usuario.

   

MySQL

MySQL es un sistema de gestión de bases de datos relacionales de código abierto, que se utiliza para almacenar y gestionar datos. En otras palabras, es un software que permite crear, manipular y consultar bases de datos, las cuales almacenan la información en tablas con filas y columnas.

   

Java

Java es un lenguaje de programación de propósito general, orientado a objetos y multiplataforma. Fue desarrollado por Sun Microsystems (ahora parte de Oracle) y es ampliamente utilizado para crear diversas aplicaciones, desde software empresarial hasta aplicaciones móviles y web.

   

Recomendaciones para que aprendas

Tailwind

Tailwind CSS es un framework de CSS de bajo nivel que proporciona un conjunto de clases de utilidad para construir rápidamente interfaces de usuario personalizadas.

Git

Git es un sistema de control de versiones distribuido que permite a los desarrolladores realizar un seguimiento de los cambios en sus proyectos de software.

Github

GitHub es una plataforma en la nube que sirve como un repositorio para almacenar y compartir código, especialmente para proyectos de software. Permite a los desarrolladores colaborar, gestionar versiones de código, y compartir su trabajo con otros.

bootstrap

Bootstrap es un framework de código abierto de front-end que facilita el desarrollo de sitios web responsivos y de aspecto consistente. Proporciona componentes y estilos CSS, HTML y JavaScript predefinidos que los desarrolladores pueden usar para construir interfaces de usuario de manera más rápida y eficiente.

Figma

Figma es una herramienta de diseño basada en la web que permite crear, compartir y probar diseños de interfaces de usuario (UI) para sitios web, aplicaciones móviles y otros productos digitales.
