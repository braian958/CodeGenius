Html

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).

Css

"Cascading Style Sheets" o "Hojas de Estilo en Cascada",Usado para definir la presentación visual de una página web, separando el contenido del diseño. Permite controlar elementos como colores, fuentes, tamaño, espaciado, entre otros, para crear una apariencia visual consistente y atractiva.

Java script

es la extensión de archivo más común para los archivos de código JavaScript. Estos archivos contienen instrucciones que los navegadores web interpretan para crear páginas web interactivas y dinámicas. Los archivos .js son esencialmente archivos de texto plano que pueden ser editados con cualquier editor de texto y se ejecutan en el navegador del usuario.

   

MySQUL

MySQL es un sistema de gestión de bases de datos relacionales de código abierto, que se utiliza para almacenar y gestionar datos. En otras palabras, es un software que permite crear, manipular y consultar bases de datos, las cuales almacenan la información en tablas con filas y columnas.

   

LPP

Lpp es para principiantes, el cual fue diseñado con la idea de facilitar el proceso de enseñanza-aprendizaje de un leguaje de programación en nuestro idioma, este contiene la mayoría de instrucciones que tienen los lenguajes de programación .
